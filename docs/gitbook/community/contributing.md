# Contributing

Limer's Capital is open source and welcomes contributions from developers, educators, and Caribbean community members.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Cuzo1ace/LimersCapital.git
cd LimersCapital

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app runs at `http://localhost:5173` by default.

## How to Contribute

- **Bug reports** -- open an issue on [GitHub](https://github.com/Cuzo1ace/LimersCapital/issues) with steps to reproduce
- **Feature requests** -- open an issue describing the use case and proposed solution
- **Code contributions** -- fork the repo, create a branch, submit a pull request
- **Education content** -- suggest new lessons, quiz questions, or glossary terms
- **Translations** -- help localize content into Spanish, French, or Caribbean Creole

## Code of Conduct

All contributors are expected to:

- Be respectful and constructive in all interactions
- Focus feedback on the work, not the person
- Respect the project's mission of Caribbean financial inclusion
- Follow existing code style and patterns
- Write tests for new functionality
