# Learn -- Education System

Limer's Capital education system transforms crypto-curious Caribbean users into confident DeFi participants through structured, gamified learning.

## Curriculum Structure

The curriculum spans 8 modules across 3 difficulty levels:

| Level | Modules | Lessons | Focus |
|-------|---------|---------|-------|
| **Basics** | 3 | 14 | Blockchain fundamentals, wallet setup, Caribbean market overview |
| **Intermediate** | 3 | 13 | Solana DeFi mechanics, token swaps, LP strategies |
| **Advanced** | 2 | 10 | RWA tokenization, perpetuals, Universal Basic Ownership |

## Key Features

- **37 lessons** with progressive difficulty and prerequisite chaining
- **8 server-validated quizzes** -- answers never leave the backend, preventing client-side cheating
- **Glossary** with 100+ terms localized for Caribbean context (e.g., explaining DeFi through TTSE analogies)
- **Skill map** -- visual progress tree showing completed, available, and locked lessons
- **Progress tracking** -- completion percentages per module, overall curriculum progress, and time spent

## XP and Rewards

Every completed lesson awards XP and Limer Points (LP):

| Action | XP Earned | LP Earned |
|--------|-----------|-----------|
| Complete a lesson | 50 XP | 25 LP |
| Pass a quiz (first attempt) | 200 XP | 100 LP |
| Pass a quiz (retry) | 100 XP | 50 LP |
| Finish a module | 500 XP bonus | 250 LP bonus |

XP feeds into the platform's [10-tier ranking system](earn.md), while LP converts to $LIMER tokens at launch.
